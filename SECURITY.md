# Security Policy

## Supported versions

| Version | Supported |
| ------- | --------- |
| 2.x     | Yes       |
| 1.x     | No        |

## Reporting a vulnerability

Please **do not** open a public GitHub issue for security problems.

Instead, report them privately via GitHub Security Advisories on the repository, or contact the maintainer through their GitHub profile.

Include:

- Description of the issue
- Steps to reproduce
- Potential impact

We aim to acknowledge reports within a reasonable timeframe.
